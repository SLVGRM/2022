## Заголовок
Текст без форматирования

## Условие
<pre class="description
<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <title>Code rendering</title>
    <style>
        div {
            font-size: 24px;
        }
    </style>
</head>

<body>
    <div>Text</div>
</body>

</html>
</pre>
Отрисовать HTML код в браузере.

## Основные требования:

- Экранируйте все угловые скобки в коде и вставьте его на HTML страницу с сохранением форматирования
- Обратите внимание, что при отрисовке тега `pre` учитывается количество пробелов в HTML
- HTML и стили менять не нужно
