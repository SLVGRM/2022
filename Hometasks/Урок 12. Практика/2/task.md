## Заголовок
Выравнивание текста

## Условие
Начальный HTML код находится в папке `start`. HTML должен оставаться неизменным. Для добавления стилей используйте **styles.css**

## Основные требования:

- Задайте документу шрифт `Verdana`
- Текст хедера должен быть 74px, занимать половину ширины страницы и располагаться по центру по горизонтали. Должен обрезаться с помощью троеточия
- Задайте элементу `<div>` высоту 200px и цвет фона coral
- Задайте тексту внутри `<div>` белый цвет, отрисуйте его заглавными буквами, отцентрируйте его по горизонтали
- Задайте тексту внутри `<div>` высоту строки 200px
