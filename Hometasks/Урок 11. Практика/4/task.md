## Заголовок
Tooltip

## Условие
Создайте тултип, который будет появляться при наведении курсора мыши на текст.
Результат должен быть как на картинке `layout.png`картинке`

## Основные требования:

- Стартовый HTML в редакторе ниже
- Нормализируйте CSS
- Тултип сделайте тегом `<span>` c классом `tooltip` а основной текст тегом `<div>` c классом `text`
- Сделайте внешний отступ для `<div>` 16px. Добавьте нижнее подчеркивание `1px dotted black`. Обратите внимание на ширину текста.
- Тултип должен иметь `grey` фон, белый цвет текста, 4px радиус границы, внутренние отступы 8px, отступать вниз от текста **Hover over me** (`bottom: -40px;`)
