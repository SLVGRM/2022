## Заголовок
Image Gallery

## Условие
Начальный HTML код находится в папке `start`. HTML должен оставаться неизменным. Для добавления стилей используйте **styles.css**


- Картинки должны располагаться по горизонтали в одну линию. Если картинки не помещаются по ширине, то должен появиться только горизонтальный скролл. Пока картинки помещаются по ширине на экран, скролла быть не должно
- Каждой картинке дайте внешний отступ 8px
- На всей странице скролла быть не должно
- Курсор мышки при наведении на галерею должен быть `all-scroll`
