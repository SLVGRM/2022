## Заголовок
Установить изображение как фон элемента

## Условие
Начальный HTML код находится в папке `start`. Его можно модифицировать. Для добавления стилей используйте **styles.css**

## Основные требования:
- Нормализовать стили с помощью **normalize.css**
- Элененту `<div>` необходимо задать ширину 300px и высоту 500px
- Установить картинку `nature.jpeg` как фон, при этом картинка должна размещаться по центру и обрезаться по горизонтали (справа и слева). По высоте картинка должна точно соответствовать высоте элемента
- Тексту задать:     
   - Размер шрифта 32px
   - Насыщенность текста равную 700
   - Цвет шрифта должен быть белым
   - Отступ сверху 32px
   - Разместить текст по центру блока (`text-align: center;`)
   - Использовать `border-box`
