## Заголовок
Градиент как цвет текста 

## Условие
Задача - закрасить текст с помощью градиента

Начальный HTML код находится в папке `start`. Его можно модифицировать. Для добавления стилей используйте **styles.css**
Помним, что у этого приема ограниченная поддержка браузеров, которую можно глянуть на `https://caniuse.com/#search=background-clip` 

## Основные требования:

- Нормализуем CSS
- Обнулить внешние отступы и задать внутренние 16px для заголовка
- Разместить текст в центре по горизонтали и отобразить в UPPERCASE
- Шрифт - **Arial**, насыщенность - 700, размер шрифта - 96px
- Цвет шрифта должен быть градиентом от желтого к красному под углом 45 градусов
