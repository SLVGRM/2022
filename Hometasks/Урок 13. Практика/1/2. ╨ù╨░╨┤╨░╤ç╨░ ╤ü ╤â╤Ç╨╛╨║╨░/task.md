## Заголовок
Спрайты изображений 

## Условие
Использовать изображения `sprites.png`, чтобы изменять иконку при наведении мыши на элемент

Основные требования:

- Добавьте на страницу `<div>` с классом `icon`
- Нормализовать стили с помощью **normalize.css**
- Сделайте наружные отступы 100px для элемента
- Используйте картинку Youtube из первой строчки по умолчанию
- Используйте картинку Youtube из второй строчки при наведении
